Introduction

Build modern Web Apps with common/jqwidgets and Blazor. common/jqwidgets Blazor UI components will help you to build perfect looking web applications. Use the most advanced UI framework for Blazor and save your time for the business logic. Choose from more than 60 UI components including Grid, Chart, Scheduler, Editor, TreeGrid and more.

Highlights

Blazor is a development platform for building mobile and desktop web applications. It provides a way to build apps for any deployment target by reusing existing code. Using HTML as the template language, Blazor offers developers the possiblity to create their own components. We created multiple components by using Blazor and the common/jqwidgets framework. The single file components for all Blazor UI components are located in the common/jqwidgets-blazor folder of the download package.
