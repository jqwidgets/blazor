Styling and Appearance

jQWidgets uses a pair of css files - jqx.base.css and jqx.[theme name].css. The base stylesheet creates the styles related to the widget's layout like margin, padding, border-width, position. The second css file applies the widget's colors and backgrounds. The jqx.base.css should be included before the second CSS file.

Below is the list of CSS classes used by jqxDropDownList.
  • jqx-widget - applied to the dropdownlist.
  • jqx-dropdownlist-content - applied to the dropdownlist's content element which displays the selected item's text or html.
  • jqx-dropdownlist-content-disabled - applied to the dropdownlist's content element which displays the selected item's text or html and the widget is disabled.
  • jqx-dropdownlist-state-normal - applied to the dropdownlist in default state.
  • jqx-fill-state-normal - applied to the dropdownlist in default state.
  • jqx-dropdownlist-state-hover - applied to the dropdownlist in hovered state.
  • jqx-fill-state-hover - applied to the dropdownlist in hovered state.
  • jqx-dropdownlist-state-selected - applied to the dropdownlist in selected state.
  • jqx-fill-state-pressed - applied to the dropdownlist in selected state.
  • jqx-dropdownlist-state-disabled - applied to the dropdownlist in disabled state.
  • jqx-fill-state-focus - applied to the dropdownlist when it is focused.
  • jqx-fill-state-disabled - applied to the dropdownlist when it is disabled.
*jqxDropDownList internally use the jqxListBox widget. For additional information about styling jqxListBox, please take a look at the jqxListBox's Styling and Appearance help topic.

When you create a custom style with colors and backgrounds for jqxDropDownList, you need to do the following:
  • Add the above CSS classes related to jqxDropDownList
  • After each CSS class, add your theme name.
    For example:
    jqx-dropdownlist-summer
  • To apply your custom style to jqxDropDownList, you need to set its 'theme' property(option) to point to your theme name string.
