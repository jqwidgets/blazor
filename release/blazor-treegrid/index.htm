Blazor TreeGrid UI Component

The TreeGrid for Blazor is a component used to present hierarchical data in a tree like grid structure. It supports multi column display of hierarchical data, data paging, sorting and filtering, data editing, columns resizing, fixed columns, conditional formatting, aggregates and rows selection. The TreeGrid can be bound to Local Data, XML, JSON, CSV, TSV and JSONP. It can operate with large data sets in paging mode.
