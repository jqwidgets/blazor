Blazor TreeMap UI Component

The TreeMap component for Blazor displays hierarchical data as a set of nested rectangles. Each element of the tree is given a rectangle, which is then tiled with smaller rectangles representing sub-branches. A leaf node's rectangle has an area proportional to a specified dimension of the data. The colors of the different rectangles can be customized.
