Blazor Scheduler Component

The Scheduler for Blazor is an advanced UI component enabling outlook style events scheduling. The component has rich functionality including Timeline, Agenda, Day, Week and Month views. You can bind the Scheduler component to JSON and ICalendar.

Appointments customization

The Scheduler enables appointment statuses, restrictions and customization of colors and appearance.

Time customization and complex features

For working across time zones the Scheduler component allows time zone visualization and time scales configuration. Complex Blazor applications can benefit from advanced features like context menus, timeline views with custom slot width, Row height configuration, week numbers view, localization, right to left support and printing.
